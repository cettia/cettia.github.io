Cettia

A real-time application framework

Cettia is a real-time application framework that allows you to focus on event handling itself by providing full duplex connection and elegant patterns.

Explore the Cettia subprojects:

  • Cettia Protocol
  • Cettia Java Server
  • Cettia JavaScript Client

Event handling

All you need is to write event handlers.

{% capture panel %} ```java // When a socket is created server.onsocket(socket -> {}); ``` {% endcapture %}{{ panel | markdownify }}
{% capture panel %} ```java // Runs for every socket in the server server.all(socket -> {}); ``` {% endcapture %}{{ panel | markdownify }}
{% capture panel %} ```java // When a socket sends some event socket.on("echo", data -> {}); ``` {% endcapture %}{{ panel | markdownify }}
